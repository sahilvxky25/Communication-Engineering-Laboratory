# Communication Engineering Laboratory

This repository contains MATLAB Live Scripts (`.mlx`) for common communication engineering laboratory experiments and simulations.

## Repository Contents

### Analog Communication
- `AM Modulation.mlx`
- `AM Simulation.mlx`
- `AM Radio Station.mlx`
- `AM Transmitter and Receiver.mlx`
- `AM modulator and demodulator Full AM  SSB  DSBSC  SSBSC.mlx`
- `FM Modulation.mlx`
- `FM Modulation with different modulating Signal.mlx`
- `FM Simulation.mlx`
- `FM Radio Station.mlx`
- `FM Transmitter and Receiver.mlx`

### Digital Communication
- `BPSK.mlx`
- `QPSK.mlx`
- `FSK.mlx`
- `Pulse Code Modulation.mlx`
- `Delta Modulator.mlx`
- `TDM.mlx`
- `Base Band.mlx`

### Synchronization and Recovery
- `Carrier Recovery System.mlx`
- `Clock Recovery system.mlx`

## Requirements

- MATLAB (recommended: version with Live Editor support)
- Basic understanding of communication systems and signal processing

## How to Use

1. Clone or download this repository.
2. Open MATLAB.
3. Open any `.mlx` file from the repository.
4. Run the script section-by-section or fully from the Live Editor.
5. Modify parameters to observe behavior under different conditions.

## Notes

- Scripts are organized as individual lab exercises.
- File names correspond to experiment topics and can be used independently.
